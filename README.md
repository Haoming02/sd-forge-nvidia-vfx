# SD Forge Nvidia VFX
This is an Extension for [Forge Neo](https://github.com/Haoming02/sd-webui-forge-classic/tree/neo), which implements **VideoSuperRes** from [nvidia-vfx](https://pypi.org/project/nvidia-vfx/) to achieve blazingly fast high quality **image upscale**

> [!Important]
> Only supports Nvidia GPU
